# VELYX v0.5 Public Preview Release Notes

Florynx Labs is thrilled to announce the **VELYX v0.5 Public Preview** — a complete compiler-first developer platform!

---

## 🌟 What's New in v0.5

### 1. Official Platform Website & Interactive 4-Panel Playground (`velyx.dev`)
* **Live Synchronized 4-Panel Compiler Playground:** Observe `.vx` source code transform in real time into Template AST, Intermediate Representation (IRRoot), emitted JS, and live DOM sandbox rendering.
* **Linear / Vercel Dark Mode Design System:** Sleek aesthetic, responsive navigation, benchmark matrix, and step-by-step interactive tutorials.

### 2. Official VS Code Extension (`extensions/vscode-velyx`)
* Native `.vx` syntax highlighting for `<template>`, `<script>`, and `<style>` blocks.
* Autocompletion snippets (`vx-component`, `vx-state`) and hover information provider for directives.

### 3. DevTools Visual Inspector Overlay (`@velyx/devtools`)
* Floating in-app DevTools inspector overlay tracking live signals, active subscriptions, and component tree metrics.

### 4. Reproducible Framework Benchmark Comparison (`benchmarks/framework-comparison.ts`)
* Matrix comparing VELYX against SolidJS, Svelte 5, Vue 3, and React 19 across bundle size (2.4 KB gzip), signal update latency (0.002 ms), and SSR HTML render throughput.

### 5. Community Governance & Starter Templates
* Added `starter-minimal`, `starter-blog`, `starter-dashboard`, and `starter-ssr` templates.
* Published official RFC process ([docs/rfcs/0001-rfc-process.md](file:///d:/projets/velyx/docs/rfcs/0001-rfc-process.md)), Brand Guidelines ([docs/brand/guidelines.md](file:///d:/projets/velyx/docs/brand/guidelines.md)), and GitHub issue templates.
