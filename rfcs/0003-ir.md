# RFC 0003: Compiler Intermediate Representation (IR) Pipeline

- **RFC**: 0003
- **Author**: Florynx Labs Core Team
- **Status**: Accepted
- **Created**: 2026-07-22

## Summary
Introduces a decoupled Intermediate Representation (IR) layer into `@velyx/compiler` to decouple SFC template parsing from backend code generation target drivers (DOM, SSR, Native, Custom).

## Compiler Pipeline

```text
.vx File → SFC Parser → AST → IR Transformer → Optimization Passes → Codegen → Generated JS Module
```

## IR Node Structure
- `IRElementNode`: HTML/Custom component node.
- `IRTextNode`: Static or reactive interpolated text node.
- `IRDirectiveNode`: Event, model, or structural directive (`vx-click`, `vx-if`, `vx-for`).
- `IRScopeNode`: Reactive scope encapsulation boundary.
