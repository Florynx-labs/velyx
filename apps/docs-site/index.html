Architectural Highlights

      
Designed from first principles to outpace traditional framework runtimes.

    


    

      

        
⚡

        
Compiler-First & Zero-VDOM

        
Dynamic expressions are isolated statically at compile time. No Virtual DOM diffing overhead or reconciler memory pressure.

      


      

        
💎

        
Fine-Grained Signal Graph

        
Reactivity operates on exact DOM text nodes and attributes. State updates trigger surgical updates without re-running component functions.

      


      

        
⏱️

        
Priority-Based Scheduler

        
Time-sliced update queue prioritizing user interactions over background background work for smooth 60fps frame rates.

      


      

        
📦

        
Tiny Runtime Footprint

        
Core runtime library is less than 3KB gzipped. Unused primitives are automatically tree-shaken by Vite or Rollup.

      


      

        
🛠️

        
Complete Developer Platform

        
Official Vite plugin, VS Code extension with syntax highlighting, DevTools browser overlay, and CLI project generator.

      


      

        
🔒

        
Ultra-Strict TypeScript Core

        
Built with 100% strict TypeScript types (`noUncheckedIndexedAccess`, `exactOptionalPropertyTypes`) across the monorepo.
