# VELYX — Official VS Code Extension

Official language support, syntax highlighting, snippets, and `.vx` file icons for **VELYX** — the compiler-first reactive frontend platform created by **Florynx Labs**.

---

## ✨ Features

- **Syntax Highlighting**: Complete TextMate grammar highlighting for `.vx` Single-File Components (`<template>`, `<script>`, `<style>`, `state`, `vx-*` directives, `{{ mustache }}`).
- **Custom File Icons**: Instant recognition for `.vx` files in the VS Code File Explorer and Tab Bar.
- **Autocompletion Snippets**: Quick scaffolding with `vx-component` and `vx-state` snippets.
- **Hover Documentation**: Detailed tooltips when hovering over `state` declarations or `vx-*` compiler directives.

---

## 🚀 Quick Start

1. Install the extension in VS Code or Open VSX.
2. Open any `.vx` Single-File Component.
3. Enjoy fine-grained compiler-first reactivity editing!
